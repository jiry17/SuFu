Module Smtlib.SmtLib

This module implements an interface to the SMT-LIB standard. Some parts of documentation will refer to a specific section of the SMT-LIB standard available online (http://smtlib.cs.uiowa.edu/papers/smt-lib-reference-v2.6-r2021-05-12.pdf).

module OC = Stdio.Out_channel
module IC = Stdio.In_channel
type numeral = Base.int

The type of numbers.

type smtSymbol =
| SSimple of Base.string
| SQuoted of Base.string

A SMT symbol is either a string identifier (e.g. x) or a quoted symbol (e.g. | this|). See Section 3.1 of the SMT-LIB language standard.

val symb_equal : smtSymbol -> smtSymbol -> bool
val str_of_symb : smtSymbol -> Base.string
type smtIndex =
| INum of numeral
| ISym of smtSymbol

An SMT index is either a number or a symbol (Section 3.3).

type smtIdentifier =
| Id of smtSymbol
| IdC of smtSymbol * smtIndex Base.list(*

For example, (_ vector-add 4 5) is an indexed identifier.

*)

A SMT identifier is either a symbol or an indexed symbol (Section 3.3).

type smtSort =
| SmtSort of smtIdentifier(*

For example SmtSort(IdC(SSimple("BitVec"), INum(3))) is (_ BitVec 3)

*)
| Comp of smtIdentifier * smtSort Base.list(*

For example Comp(Id(SSimple("Set")),SmtSort(Id(SSimple("Int")))) is (Set Int)

*)

A SMT Sort (a type) is an identifier or a composite type.

type smtSortedVar = smtSymbol * smtSort

A SMT sorted var is a symbol associated with a sort.

type smtQualIdentifier =
| QI of smtIdentifier
| QIas of smtIdentifier * smtSort

A qualified identifier is either a simple identifier QI ... or an identifier qualified with a sort QIas(x,y) is (as x y).

type smtAttribute =
| ASymb of Base.string
| ASymbVal of Base.string * Base.string

An SMT attribute that is either a string symbol or a symbol with a string value.

type smtPattern =
| Pat of smtSymbol
| PatComp of smtSymbol * smtSymbol Base.list

A SMT pattern is either a variable (a SMT symbol) or a constructor (a pair of a constructor name with a lsit of arguments).

type smtSpecConstant =
| SCNumeral of numeral
| SCDecimal of numeral * Base.int * numeral
| SCHexaDecimal of Base.string
| SCBinary of Base.bool Base.list
| SCString of Base.string

The type of SMT special constants (Section 3.1).

type solver_response =
| Error of Base.string
| Sat(*

The solver responded sat

*)
| Unsat(*

The solver responded unsat

*)
| Unknown(*

The solver responded unknown

*)
| Success(*

The solver responded success (the :print-success option is on)

*)
| Unsupported(*

The solver reponded that the input is not supported.

*)
| SExps of Base.Sexp.t Base.list(*

Any other type of response is a list of S-Expressions, which can be interpreted later.

*)
val parse_response : Base.Sexp.t Base.list -> solver_response
