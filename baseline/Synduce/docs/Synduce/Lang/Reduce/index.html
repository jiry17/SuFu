Module Lang.Reduce

val until_irreducible : ( Term.term -> Term.term * bool ) -> Term.term -> Term.term

until_irreducible f t applies f to t until t is unchanged by application of f or we have reached the limit of rewrites Reduce._MAX.

  • parameter f

    A function that returns a pair of a term and a boolean. It should be written such that f x = x', b => (b = (x != x')). The boolean indicates whether the term has be changed.

  • parameter t

    The term that will be rewritten using f.

val rule_lookup : Term.Context.t -> ( 'a, Term.variable * Term.variable list * Term.pattern option * Term.term, 'b ) Base.Map.t -> Term.variable -> Term.term list -> Term.term list

rule_lookup ctx rule_map f args searches for a rewrite rule in rule_map whose head matches f args. For each matched rule, it returns the rhs of the rule with the substitutions corresponding to apply f args applied. If no rule is matched, returns an empty list.

val reduce_term : ?projecting:bool -> ?unboxing:bool -> fctx:PMRS.Functions.ctx -> ctx:Term.Context.t -> Term.term -> Term.term

reduce_term ~fctx ~ctx t reduces the term t using lambda-calculus reduction rules. (let x = e in e' is equivalent to (λx.e') e). Function symbolds in applications are resolved by looking up functions in the current function contexts fctx or context ctx.

val reduce_pmrs : fctx:PMRS.Functions.ctx -> ctx:Term.Context.t -> PMRS.t -> Term.term -> Term.term

reduce_pmrs ~fctx ~ctx p t is a shortcut for reduce_term ~fctx ~ctx (mk_app (mk_var p.pmain_symb) [t]) when p is in the current environment.

val calc_term : fctx:PMRS.Functions.ctx -> ctx:Term.Context.t -> Term.term -> Term.term list

Same as `reduce_term` but returns a list of terms, one for each reduction step using a pmrs rule.

val instantiate_with_solution : fctx:PMRS.Functions.ctx -> ctx:Term.Context.t -> PMRS.t -> (string * Term.variable list * Term.term) list -> PMRS.t

]instantiate_with_solution ~fctx ~ctx p defs] returns the PMRS p, in which the unknowns have been replaced by their definition defined in defs.

  • parameter fctx

    The function context.

  • parameter ctx

    The current context with types and names.

  • parameter p

    A PMRS with unknowns.

  • parameter defs

    Definitions for the unknown in the PMRS, in the form of a list of triples (unknown function name, arguments, body of the unknowns)

val is_identity : fctx:PMRS.Functions.ctx -> ctx:Term.Context.t -> PMRS.t -> bool

Returns true if the PMRS is equivalent to the identity function. Uses the reduction procedures to check wether f x = x, symbolically.
