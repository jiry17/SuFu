Infer the logic to use in calls using terms.

  • parameter quantifier_free

    is true by default, set to false if quantifiers will be used in formulas.

  • parameter with_uninterpreted_functions

    is false by default, set to true if uninterpreted functions will be used.

  • parameter hint

    is an optional logic hint. The current implementation returns exaclty this hint but this will change.

  • parameter logic_infos

    a list of logic_infos, possibly from context program
