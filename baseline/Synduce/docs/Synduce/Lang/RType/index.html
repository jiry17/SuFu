val fun_typ_unpack : t -> t Base.list * t

fun_type_unpack t uncurries the type t as a function. Returns a pair of a list (the type of the arguments of the function) and a type (the output type of the function). If t is not a function type, then (,t) is returned.

val fun_typ_pack : t Base.list -> t -> t

fun_typ_pack args t construct the type of a function args -> t: for example fun_typ_pack [t1;t2] t_out is TFun(t1, TFun(t2, tout)).
